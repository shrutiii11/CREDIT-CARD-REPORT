Credit Card Analytics Dashboard
An interactive data analysis project built using Power BI to explore credit card transactions and customer behavior. The dashboard provides insights into revenue trends, spending patterns, and customer segmentation to support data-driven decision-making.

🚀 Project Objective
To analyze credit card transaction data and identify key business insights related to:
1.Revenue performance
2.Customer demographics
3.Spending behavior
4.Financial trends

📈 Key Insights
Total revenue exceeded 55M, with strong transaction volume
Swipe transactions contribute the highest share of revenue
Blue card category dominates overall revenue generation
Business and white-collar professionals are top revenue contributors
Revenue varies significantly across income groups and customer segments

📊 Dashboard Features
X Revenue analysis by:
   1.Expenditure type (Bills, Fuel, Travel, etc.)
   2.Card category (Blue, Silver, Gold, Platinum)
   3.Customer attributes (education, job, income)
X Weekly revenue trends with WoW (Week-on-Week) comparison
X KPI tracking:
   1.Total Revenue
   2.Transaction Amount
   3.Interest Earned
X Customer segmentation by:
  1.Age group
  2.Gender
  3.Marital status
